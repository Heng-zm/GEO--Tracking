# **App Name**: GeoTrack Now

## Core Features:

- Real-time Location Tracking: Continuously update and display the user's current GPS coordinates in real-time.
- Latitude Display: Show the current latitude with high precision.
- Longitude Display: Show the current longitude with high precision.

## Style Guidelines:

- Primary color: Soft gray (#B3B3B3) to subtly complement the bright white text, aiming for balance without distraction.
- Background color: Black (#000000) to ensure maximum contrast and readability of the coordinates.
- Accent color: Light blue (#ADD8E6) to subtly highlight accuracy data.
- Font: 'Inter', sans-serif, for clear and legible coordinate display.
- Minimalist design with coordinates prominently displayed in a large, clear font.
- Smooth, subtle transitions when updating coordinates.
